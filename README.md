# Fruit Merge Game (Unity Project)

## 📝 Introduction
A physics-based puzzle project built with **Unity Engine**, inspired by the core mechanics of "Suika Game" (Watermelon Game). Players drop various fruits into a container and merge identical ones to create larger, higher-tier fruits.

**Note on Development:** This project was created for **educational purposes** following the curriculum of the **"Unity Suika Game"** course by **Tabsil**. The core game logic, architectural patterns, and source code are implemented based on the course materials to better understand Unity's physics system and game state management.

## 🚀 Current Features
* **Physics-based Gameplay:** Automated gravity and collision handling for all objects.
* **Merge Logic:** Core system that handles the collision of identical fruits to spawn the next evolutionary tier.
* **Spawner System:** Controls for fruit drop positions via mouse or touch input.
* **Core Game Loop:** Basic scoring system and "Game Over" state detection.

## 🛠 Tech Stack
* **Engine:** Unity 2022.3+
* **Language:** C#
* **Platform:** PC / Mobile (Android/iOS)

## 📂 Project Structure
* `/Assets/Scripts`: Contains Merge logic, Spawner controls, and Game Management.
* `/Assets/Prefabs`: Pre-configured fruit objects with physics components.
* `/Assets/Sprites`: 2D visual assets for the game.
